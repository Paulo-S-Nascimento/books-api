# 📚 Books API

API simples de livros desenvolvida em **Python puro**, sem frameworks, com **CI/CD via GitHub Actions** e **deploy automático no Render**.

---

## 🚀 Funcionalidades

- 🔍 **Listar todos os livros**
- 🔍 **Buscar um livro por ID**
- ➕ **Adicionar um novo livro**
- 🔗 **Status da API**
